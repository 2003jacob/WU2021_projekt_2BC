Vem är jag och varför gör jag detta? Mitt namn är Jacob Kristensen och jag skapar den här sidan för att göra det lättare för dom som är intresserade att skapa sina egna datapack till Minecraft moddet origins mod.

Detta är ett slut projekt i webbutveckling på NTI johanneberg årskurs två. Nyligen började jag koda ett datapack för minecraft moddet origins mod. Origins mod har en dedicarad discord server för frågor. När jag passade på att hjälpa några som behövde hjälp på server upptäckte jag att många nybörjare hade många problem med dom grundläggande sakerna som t.ex fil struktur. Jag skapade den här sidan ihop om att hjälpa dom som är intresserade men oroliga att dom inte skulle kunna komma igång.
